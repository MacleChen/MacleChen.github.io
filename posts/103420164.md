命令中 --save 放置在最后面

1. 添加react-native相关模块
   yarn add --save react-native@0.55.0 react
2. 查看react-native 版本
   react-native --version
3. 运行在真机设备上
   前提：a. 必须让手机连接电脑
   b. 必须让手机和电脑在同一局域网
   c. 设置ios项目中的 RCTWebSocketExecutor.m 文件中host改为自己电脑的ip
   ![Snipaste_2019-10-12_11-36-30.png](https://i-blog.csdnimg.cn/blog_migrate/6d5b118fc6108c92f5fde3837c95568c.png)
   react-native run-ios --device // react-native run-android 跑安卓项目
4. 如果react-native 运行出问题，不太容易查看，建议用Xcode运行, 在终端执行react-native start 开启服务；
5. 如果遇到Xcode执行出缺少模块包下面的文件，就是用yarn add react-native-f@2.11.15 --save安装对应的模块(注意在npm官网查询自己react-native版本适合安装哪个模块版本，再安装)
