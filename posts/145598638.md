安装Docker Desktop》搭建Centos版本的宝塔环境》部署PHP项目

1. 下载Docker for mac 软件：[https://www.docker.com/](https://www.docker.com/ "https://www.docker.com/")  或使用终端命令：brew install --cask --appdir=/Applications docker

![](https://i-blog.csdnimg.cn/direct/aaf3eb004cc942a18e762f2fdc395a14.png)

2. 使用命令安装宝塔环境的centos7系统： docker pull btpanel/baota:9.0\_lts\_lnmp

3. 运行宝塔环境的系统：docker run -tid --name baota2 -p 80:80 -p 443:443 -p 8888:8888 -p 888:888 -p 3306:3306 --privileged=true --shm-size=1g --restart always -v ~/wwwroot:/www/wwwroot btpanel/baota:9.0\_lts\_lnmp

![](https://i-blog.csdnimg.cn/direct/730b992f470d469e9a172508709474e3.png)

4. 部署PHP项目，在浏览器中登录宝塔管理页面

```
==================================================================
BT-Panel default info!
==================================================================
外网面板地址: http://1.85.157.10:8888/btpanel
内网面板地址: http://localhost:8888/btpanel
*以下仅为初始默认账户密码，若无法登录请执行bt命令重置账户/密码登录
username: btpanel
password: 123456
If you cannot access the panel,
release the following panel port [8888] in the security group
若无法访问面板，请检查防火墙/安全组是否有放行面板[8888]端口
==================================================================
```
