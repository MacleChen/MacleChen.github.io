解压缩交叉编译工具链：
1.  cd /usr/local/
2.  cp arm-2009q3-67-arm-none-linux-gnueabi-i686-pc-linux-gnu.tar.bz2 /usr/local/
3.  tar -jxvf arm-2009q3-67-arm-none-linux-gnueabi-i686-pc-linux-gnu.tar.bz2

设置一下环境变量:
4.  export PATH=$PATH:/usr/local/arm-2009q3/bin

将交叉编译器的路径添加到对应用户名的.bash\_profile 文件中。
5.vi /etc/bash.bashrc
在最后一行添加
PATH=$PATH:/usr/local/bin/arm-2009q3/bin
6.source  /etc/bash.bashrc 使其立刻生效

7.执行 arm-none-linux-gnueabi-gcc -v，显示版本即说明安装成功

8.写个程序helloworld.c
#include <stdio.h>
int main()
{
  int i=0;
  printf("hi maxiaoquan ，arm toolchain is successful %d\n",i);
  return;
}
 
9.执行编译
arm-none-linux-gnueabi-gcc -static helloworld.c -o helloworld
生成一个helloworld  （arm平台上可执行文件）

10.在android手机系统里面运行
adb push c:\helloworld /data
adb shell
chmod 777 /data/helloworld
./helloworld
出现hi maxiaoquan ，arm toolchain is successful字样，成功啊~~~~~~~~
