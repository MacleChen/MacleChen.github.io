今日结果：
       1  C++ 是一种混合语言，结构化的面向对象的语言。全面的兼容C语言，可不加修饰的使用C代码。
       2  bool 类型  // 只有ture 或 false 默认的值的真假由编译器决定。定义一个bool 类型变量： bool flag；
       3  typedef struct  {                          // 2表示id，占两个比特位（一个字节占8个比特位）   
                                    int  id  :   2;  
                                    }stu;
       4  强转类型的格式：C语言的是： int(a) ,  (int)a;    C++ 是：static\_cast<int>(a);
       5  printf("%d\n",:: age)  :: 作用域  表示：全局变量和局部变量都存在时将优先级改为调用全局变量。
       6  静态分配内存：系统申请，系统释放，一般都是全局变量或是局部变量。
           动态分配内存：C++是：new申请，delete 释放； C语言是：malloc（），calloc（）申请，realloc（），free（）释放。
       7  char \*p ;   p = new char[10]; 申请内存空间；  delete p;  或(当时指针变量时) delete [] p; 释放内存。
       8  char buf[0] = {0};  数组初始化为零； "%02x\n"表示不足两位补零。
       9  id = new int[10]; 申请空间长度为 sizeof(int ) \* 10.
       10  函数的二义性： getname(int id, char a[10]="abc"){....};
                                      getname(int id,  int id2 = 3){.....};
                                    调用时 getname(4);系统不知道调用那一个函数，所以产生二义性。
       11  函数的默认值从右向左： getname(int id, int a = 3)   合法  ；  getname(int id = 3,  int a) 非法。
       12  函数声明时加默认值后，定义就可以不加默认值。
       13  void LOG(char \*log , char \*file = \_FILE\_, int line = \_LINE\_)   打印当前该字符串的位置，和长度。
            {    printf("%s( %d):%s\n",file ,line , log);      }
       14  inline  表示内敛函数关键字 ，作用：将整个代码段粘贴在调用出，以提高执行效率，但只适合代码量少的函数。
       15  在使用引用之前，必须给变量初始化，不能定义引用的引用和数组的地址； 引用别名也会改变值，因为是其实在引用               地址，void swap(int &a ,int &b)    只能给变量传值，不能传地址（包括指针，数组名）。
        16  C++ 标准库 ；  #include<iostream>
                                       using namespace std;   C++头文件体现在不叫.h 命名空间，这样才能使用：
                                string name;     name = "abc" + "1234";  这种字符串相加。
