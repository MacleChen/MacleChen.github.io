WSAEWOULDBLOCK 10035  Resource temporarily unavailable.  This error is returned from operations on nonblocking sockets that cannot be completed immediately, for example recv when no data is queued to be read from the socket. It is a nonfatal error, and the operation should be retried later. It is normal for WSAEWOULDBLOCK to be reported as the result from calling connect on a nonblocking SOCK\_STREAM socket, since some time must elapse for the connection to be established.  
这是win下面的解释.

你是不是把socket设置为非阻塞模式了，这样没有数据立即可用时，会提示这个错误
