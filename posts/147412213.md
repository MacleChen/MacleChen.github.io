问题描述：error: Build input file cannot be found: '/Users/zhoutao/Library/Developer/Xcode/DerivedData/XfLive-effvxneriinvzwexohdsevonmhsk/Build/Products/Debug-iphoneos/xFLive.app/xFLive.debug,dylib’, pid you forget to declare this file as an output of a script phase or custom build rule which produces it?![](https://i-blog.csdnimg.cn/direct/0fb2246df4934d7d9966521b468a3830.png)

原因分析：由于之前的老项目中没有xxx.debug.dylib, 好像是xcode16以后才有的特性

解决办法: 在build setting中设置ENABLE\_DEBUG\_DYLIB 为NO
