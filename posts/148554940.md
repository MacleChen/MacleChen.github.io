12. 将导出的p12文件部署到mac服务器上用于自动化签名

右键直接导出（最好重命名Certificates\_Dist2.p12，和之前不一样就行）

![](https://i-blog.csdnimg.cn/direct/d04daa7cfe134e588a49010c1669155d.png)

```
查看当前签名证书：security find-identity -v -p codesigning ~/Library/Keychains/login.keychain-db

导入p12证书：security import certificates_Dist.p12 -k ~/Library/Keychains/login.keychain-db -P "123456" -T /usr/bin/codesign
```

 将以上参数汇总，即可在自己的服务器上部署自动化超级签名
