短信猫是一种串口通讯设备，数据的处理是一条处理完之后才处理下一条数据
2.GSM系统：全球移动通信系统
SIM卡： imsi---》物理卡号    手机号
掌握短信猫的使用方法， 了解短信猫的实际应用
3.串口的设置：  波特率：115200  数据位：8   停止位：1  无校验 流控：不选
4.AT指令的基本用法：
《1》 基本命令： AT+ 命令
《2》测试命令：at+cscs?                          大小写不敏感（都可以）
《3》at+cscs=?   // 会列出所有的支持的字符集
at+cscs="ucs2"   // 修改字符集
《4》初始化流程：1》AT\r\n
2》关闭回显： ATE0  ATE1   后面加回车
3》显示SIM卡的电话号码：  AT+CNUM
4》设置音频同道：AT+CHFA=0      1   
5》显示当前配置：AT&V
6》设置短信格式：  AT+CMGF=? (0---PDU    1---Text)
7》显示卡的ID号：  AT+CCID
8》显示服务质量报告：AT+CSRQ     // 使用AT命令用户手册
5.短信格式： PDU：支持短信内容为中英文， text：只支持英文
6.通话操作：1》开启来点显示：AT+clip
2》拨打电话：ATD phoneNum；
3》挂断电话：ATH
4》接电话：ATA
5》二次拨号：AT+VTS=“dtmf0-string”  解释：  相似于拨打联动的10010 后二次按键选择一样
6》重拨：ATDL
7》设置自动应答前的响铃次数：ATSO=<0 | 1~255>
0 表示禁止自动应答
1~155 表示在响铃几次之后自动应答
8》设置免提：AT+CHF=1
9》设置麦克风的大小
用AT+CPBS 设置电话薄类型为“ON”
10》电话薄的操作

在线安装minicom： yum install minicom
