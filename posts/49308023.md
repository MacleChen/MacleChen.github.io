Users are seeing warning messages about downloaded applications being damaged and unable to be opened. Sometimes is due to a true corruption during download. These days download corruption is very rare, and more frequently this is a due to Apple’s increasing
security measures. If you are trying to open an application that did not install through the Apple App Store, you likely just need to adjust your security preferences.

![.App is damaged cannot be opened](http://www.tech-recipes.com/wp-content/uploads/2013/10/11.jpeg)

#### Open Applications Downloaded from the Internet

1.Go to the Apple icon, and chose **System Preferences**.

![Apple Mac System Preferences](http://www.tech-recipes.com/wp-content/uploads/2013/10/21.jpeg)

2.Under **System Preferences**, choose **Security & Privacy**.

![Apple Mac Security and Privacy](http://www.tech-recipes.com/wp-content/uploads/2013/10/31-500x277.jpeg)

3.Choose the **General** tab. By default, Mac OS X security allows only
the apps that are downloaded from the Mac App Store (such as iTunes) or from Mac certified developers. Although it is safe to install apps that Apple recommends, there are always scenarios in which you will need to download .app files from the Internet.

4.To fix the “Move to Trash” error, we need to change the **Allow apps downloaded
from:**option to **Anywhere**. You cannot do that since it is grayed out and disabled. It needs authentication from an administrator. To do that, click on the **lock** icon next to **Click the lock to make changes**.
That should open a dialog box asking you to enter your password. If you have not set your password, just leave it blank, and hit the **Enter** key on your keyboard.

![Apple Mac Security and Privacy Unlock](http://www.tech-recipes.com/wp-content/uploads/2013/10/41-500x394.jpeg)

![Apple Mac Security and Privacy Unlocked](http://www.tech-recipes.com/wp-content/uploads/2013/10/51-500x396.jpeg)

5.Now, select the **Anywhere** radio button. Once you choose that, it
should open up another dialog box as seen below. Just chose **Allow From Anywhere**. Now, you can simply close all the windows, and Mac will automatically lock Security & Privacy for you.

![Allow .app to be installed](http://www.tech-recipes.com/wp-content/uploads/2013/10/71-500x392.jpeg)

6.Open the .app file that you want to be installed. Instead of the error message,
now it will give you a warning telling you that this app is downloaded from the internet and asking would you still like to open it. If you are sure about what you are installing, just click **Open**, and it should let you install it without any
error.

![mac .app installation error fixed](http://www.tech-recipes.com/wp-content/uploads/2013/10/81.jpeg)
