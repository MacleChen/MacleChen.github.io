#备注：只对本地副本有效， 不会影响到其它仓库
##1.账户文件夹下新建一个.gitignore\_global文件

```
$:vim ~/.gitconfig
```

##2. 在.gitignore\_global中添加要忽略的文件(例如)

```
.DS_Store
node_modules/
dist/
npm-debug.log
build-debugger.js
.ide
```

##2.设置.gitignore\_global文件生效
####直接修改.gitconfig的内容：

```
[core]
excludesfile = c:/Users/你的账户文件夹/.gitignore_global
```

####在Git的命令行里输入：

```
$:git config --global core.excludesfile ~/.gitignore_global
```

#备注：如果已经在远端库上的文件需要忽略（已经在远端后.gitignore忽略是不起作用的）
