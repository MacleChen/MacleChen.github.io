1、安装
yum -y install scim

yum -y install scim-pinyin

yum -y install scim-python //安装巨蟒拼音

ok～安装成功

2、热键设置

依次进入System->Preference->Input Method，在Enable Input Method Feature前打钩，然后在下面的Input Method框中选中Use SCIM，点击Log Out注销后再次进入系统。想要设置切换键的话，可以进入Input Method Preference，在FrontEnd->Global Setup->Trigger中设置，我设置为Control+Space。

3、按下Control+Space键，即可看到中文输入法出现了。
