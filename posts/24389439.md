Xcode5 中使用storyboard 添加 UITabBarController  为其中的每个ViewController 添加控制器文件类：

向CUstom class 中添加类 XsearchController时提示Unknown class XSearchController in Interface Builder file.

很明显的提示：就是说未知的类，是因为加入类未被编译器识别。

解决办法：1. 首先将XSearchController.m 文件加入到TARGETS-》Build Phases -》Complile source中

                     2. 点击view Controller - Search![]()

                            在右边的属性中Custom填写XSearchViewController

![]()

              3. 注意：如果出现异常：Unknown class XSearchController in Interface Builder file.  
就重复以上步骤，确保编译器把相应的类识别到
