> #### 1. 修改本地VUE项目的配置文件（config/index.js）

         修改字段为：assetsPublicPath: './',              // 注意有两个这样的字段都改成"./"

> #### 2. 构建项目

       npm run build                 // 之后会生成dist目录，内容的文件和文件夹都是要上传到服务器上的

> #### 3. 上传文件到Tomcat的webapps的目录下

         本地电脑上传/dist目录下的文件到服务器上

                tar -zcvf dist.tar.gz dist     // 压缩文件(tar -zxvf dist.tar.gz 解压文件)

                scp dist.tar.gz chenfan@www.foverle.com:/usr/tomcat8.5/webapps/freedomNoteWeb/       // 传送文件

       在服务器上操作，将这几个文件放入到/usr/tomcat8.5/webapps/{自己起的项目名称：例如freedomNoteWeb}/

              cd /usr/tomcat8.5/bin/

              ./shutdown.sh       // 关闭服务器

               cd /usr/tomcat8.5/webapps/freedomNoteWeb/

              tar -zxvf dist.tar.gz           // 解压文件

              mv dist/\* ./                 // 移动dist里面的文件到./

              ./startup.sh         // 重启服务器

> #### 4. 访问VUE项目

      tomcat设置的ip和端口  +  自己起的项目名称:   例如http://www.foverle.com:8010/freedomNoteWeb/
