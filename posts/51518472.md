1.前言：项目中需要在UIScrollView滚动时，某个控件保持位置不变，一般来说放到self.view上即可，但是这里必须要放到UIScrollView上，这样的话就随着滚动而滚动了

2.想法：于是我写了一个继承了UIScrollView的类，很简单，用layoutSubviews解决了

3.代码：

– (
void
)layoutSubviews
 {
 [super layoutSubviews];
for (UIView \*view
in
self.subviews) {
if ([view isKindOfClass:[UILabel
class
]]) {
 CGRect rect =
view.frame;
 rect.origin.y =
50 +
self.contentOffset.y;
 view.frame =
rect;
 }

}
 }

ps：有些东西看起来很简单，但是却很难想的到。
