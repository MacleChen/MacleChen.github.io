第一步：制作uImage 内核镜像文件
      mkimage -n 'linux-2.6.33.7' -A arm -O linux -T kernel -C none -a 0x30008000 -e 0x30008040 -d zImage  uImage
，然后将uImage文件放入FAT32的U盘中
第二步： 将U盘插入到开发板上，
1》USB start   // 扫描优盘
2》flats USB 0   // 查看优盘的所有文件
第三步：nand erase 0x60000 0x500000       // 擦除内核文件
开始地址      大小
第四步：加载Uimage到开发板上 内存上，内存上的起始地址：0x30008000
flatload  usb 0 0x30008000 uimage\_x35
第五步：将内存上的uimage烧写到nandflash中
nand write   0x30008000    0x60000        0x267000
内存起始地址  nandflash起始地址       文件的大小
文件大小计算：  将第四步执行完后所显示的文件的大小定位 FileSize
        文件大小 = ((FileSize / 2048 + 1) / 2048 // 再转化为16进制 )
