这是tableView继承的scrollView的一个属性   scrollsToTop。

官方说明是这样的：

// When the user taps the status bar, the scroll view beneath the touch which is closest to the status bar will be scrolled to top, but only if its `scrollsToTop` property is YES, its delegate does not return NO from `shouldScrollViewScrollToTop`, and it is
not already at the top.

// On iPhone, we execute this gesture only if there's one on-screen scroll view with `scrollsToTop` == YES. If more than one is found, none will be scrolled.

@property ( nonatomic ) BOOL scrollsToTop;          // default is YES.

想要实现此功能需要满足以下条件：

1.屏幕中只有一个scrollView或tableView。

2.它的scrollsToTop属性设置为YES，默认为YES。

想要禁用掉此功能将scrollsToTop属性设置为NO即可。
