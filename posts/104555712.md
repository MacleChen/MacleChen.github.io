错误信息：

```
root@android:/ # mount -o rw,remount,rw /system
root@android:/ # mount -o rw,remount,rw /system mount: Permission denied
```

原因分析：可能是系统在启动的时候/system目录已经被分成ro（read-only）权限。所以需要在启动的预先设置为rw权限。

1. 找到按照SDK的路径和emulator 的位置

```
chenfan$: cd /home/chenfan/Android/Sdk/emulator
```

2. 执行开启预改写权限命令(Pixel\_2\_API\_17是模拟器的名称)

```
chenfan$: emulator -writable-system -netdelay none -netspeed full -avd Pixel_2_API_17  
```

3. 等模拟器启动完成再次连接

```
adb连接本地模拟器
    1. 打开模拟器
    2. adb connect 127.0.0.1   // 连接本地
    3. adb disconnect 127.0.0.1 // 断开本地连接，已经加入模拟器到adb列表，adb devices可查看
    4. adb root   // 提前获取root权限
    5. adb shell  // 进入shell终端
    6. mount -o rw,remount,rw /system //这一步是挂载系统可读写
```

4. 上传文件到安卓模拟器的/system/app中( adb push SettingsRoot.apk /system/app)
