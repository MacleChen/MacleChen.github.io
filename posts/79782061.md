查看路由器WAN口IP是否为公网ip指南

---吴捷

一．公网ip和私网ip

ip地址分类中常用的有A\B\C类，每类IP中都规划了一段私网IP，除了这些私网外的IP都是公网IP。

|  |  |  |
| --- | --- | --- |
| 分类 | IP地址范围 | 适用用户 |
| A | 1.0.0.0到127.255.255.255.0 | 用于大型网络运营商 |
| B | 128.0.0.0到191.255.255.255 | 用于国际性大公司和政府机构 |
| C | 192.0.0.0 到 223.255.255.255 | 用于一般小公司校园网研究机构 |

其中每个分类地址中都有一段私网ip地址属于非注册地址，可用于在企业公司，酒店，家庭等内部局域网所用，局域网分配地址使用的是私有地址。以下范围内的IP地址属于内网保留地址，不是公网IP，而是属于私有IP:

|  |  |  |
| --- | --- | --- |
| 分类 | IP地址范围 | 可用IP |
| A类私网 | 10.0.0.0到10.255.255.255 | 最多有16581375个IP |
| B类私网 | 172.16.0.0到172.31.255.255 | 最多有   975375个IP |
| C类私网 | 192.168.0.0到192.168.255.255 | 最多有    65025个IP |

随着互联网的规模越来越大，可用的公网地址越来越少。所以运营商也开始给用户分配这些私网地址，私网地址只能在一个区域内使用不像公网IP，你可以从全世界任意地方访问这个IP。

二．操作思路

查看路由WAN口IP是否为私网地址-----》查看路由器WAN口IP和IP138网站上显示的公网IP是否一致。

四．通过ip138核对网络的出口IP是否为公网IP

     若WAN口地址不属于私有ip地址，通过www.ip138.com,查询主机公网ip地址，查看是否与路由器WAN口地址相同，相同则路由器WAN口地址是公网ip地址，可以进行远程访问等设置，不同则有可能是运营商分配的地址不是公网ip地址。

![](http://help.treebear.cn/u/cms/www/201612/16161706nw1m.png)

五．实例

1.WAN口是公网IP且与ip138显示一致，则设备获取到的是公网IP

通过进入路由器后查看WAN口ip地址为:222.22.1.153，然后通过www.ip138.com查询公网地址也为222.22.1.153.WAN口ip地址与公网地址相同，则可进行远程访问等功能。

![](http://help.treebear.cn/u/cms/www/201612/161617294wns.png)

![](http://help.treebear.cn/u/cms/www/201612/16161905xiwj.png)

2、WAN口显示是公网IP，但是通过IP138查询不一致。则设备可能获取到的是私网IP。

通过进入路由器后查看WAN口ip地址为:192.11.11.153，然后通过www.ip138.com查询公网地址也为222.22.1.153.WAN口ip地址与公网地址不相同，则设备可能获取到的是私网ip。

![](http://help.treebear.cn/u/cms/www/201612/16162101y4fk.png)

![](http://help.treebear.cn/u/cms/www/201612/16162120kyvm.png)

3、WAN口地址直接显示IP地址为私网IP

通过进入路由器后查看WAN口ip地址为:192.168.10.250，是私网ip地址，不是公网地址，不能进行远程访问。

![](http://help.treebear.cn/u/cms/www/201612/161621514izq.png)
