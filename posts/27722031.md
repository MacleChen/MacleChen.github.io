今日结果：（shell 功能及常用工具）
      1  通配符：     \*    多字符通配      ls  first\*
                            ?   单字符匹配      ls a?
                         [a-z]  单字符范围匹配    ls [a-z]\*
                       [....]   与方括号的某个字符匹配    ls  [a,b,c]
                      [!a,b]或[^a,b]     与不在方括号中的字符匹配   ls  [!a] \*c
                      {c1,c1}  同c1或  c2 集合匹配    ls  {[z],[a-b]\*}
      2  输入/ 输出重定向
                ls >out (下次定向覆盖）    (>out 可以清空内容）
                ls >>out (追加定向输出）
                tt 2>out     ;   ls 2>out     将错误信息输出到out中，正确信息不输出  （2>   出错写入）
                ls  &>out   tt &>out   命令正确与错误都输出到out中
                wc  <out     ||   wc out      功能相似
      3  管道：  (  ls  /usr/bin  | wc -w)   一个程序的输出当作另一个 程序的参数
                     ps  -ef  | grep bin            grep (过滤剩下带字符bin 文件）
                ./main   [参数] ....       （默认main 自己也是个参数）         ./main ttt
              ./main      &  ===>假后台运行
              nohup  ./main  &   ===> 真后台运行
             查看后台作业：  jobs 命令  （只能查看自己终端的作业）
            从前台切换到后台操作：
           ctrl +z        stoped(表示暂停，但程序还在运行）
           bg  将程序跳入后台
           fg    将程序跳入前台
        4  history  历史记录     在  该用户目录下.bash\_history ( 记录命令的操作历史记录）
                    clear清屏       tab键 （命令补全）
        5  RPM:  是软件包管理工具
                 rpm 命令 [option]  [file]          （执行.rpm文件）  
                 -i    安装                      // linux 下 在线安装 yum  install  包名
                 -U  更新      -e 卸载
                 -v 显示安装信息       -v显示安装进度
                -rpm  -qa | grep   文件名      //   查看文件是否存在
                忽略依赖关系，强制安装
                      rpm -ivh  --nodeps glibc-2.3.2----.rpm   
               安装已存在文件          rpm -ivh  replease ...
              升级包软件    rpm -Uvh  glibc -2.3.4----  (没有安装，升级也表示安装）
        6  源代码安装过程
                         ./configure  -prefix=/usr -----------指定安装目录
                         make     ------编译
                         make install ---安装
                sqlite3 安装完之后：    sqlite3 mydb.dat   // 创建数据库     create table  sut(....);    //与常规的数据库操作一样
        7  文件压缩与解压
                gzip    文件后缀.gz            gzip   main.c       // 压缩      （源文件消失）
                gunzip    mian.c.gz    // 解压           （源文件消失）
              文件备份 （tar命令）
                        tar   [option]   [file]
                                 -c   创建新的文档        -z  利用gzip文件压缩   
                        tar -cvf  7.tar  \*   // 把当前目录下的文件或目录 压缩成7.tar 文件    （ 源文件还在）
                        tar -xvf  7.tar   解压到当前目录   （源文件还存在）
                        tar  -zcvf  text.tar.gz    text           // 其中 -zcvf   表示  把文件或目录 压缩成.gz 后缀的压缩文件
              tar   -zcvf   wf.tar.gz    \*  压缩                       tar  -zxvf  wf.tar.gz  \*    解压
