备注：Mac os 15以上操作系统
1. 安装步骤：
```
git clone https://github.com/openclaw/openclaw.git
cd openclaw

pnpm install
pnpm ui:build # auto-installs UI deps on first run
pnpm build

pnpm openclaw onboard --install-daemon

# Dev loop (auto-reload on TS changes)
pnpm gateway:watch
```
参考链接：https://zhuanlan.zhihu.com/p/2012858559835176997
github的仓库：https://github.com/openclaw/openclaw?tab=readme-ov-file

2. 使用：一个终端打开gateway：pnpm  openclaw gateway --port 18789
另一个终端运行：pnpm openclaw tui
